About NgBookLibrary

NgBookLibrary is a simple, modular Angular application for managing a book library. It allows you to view, add, edit, and delete books, with features like pagination, notifications, and error handling. The project demonstrates best practices in Angular architecture, dependency injection, and user experience.

Key Features:

  • Paginated book list with loading and empty states
  • Add and edit books using Reactive Forms
  • Delete books with confirmation modal
  • Success and error notifications for user actions
  • Global HTTP error handling and logging
  • SOLID principles and Angular best practices
